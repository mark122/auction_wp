A simple light weight plugin to Convert the time from one timezone to other timezone.

== Description ==

This is a light weight plugin which allows your customer to convert the time from one timezone to another timezone.

Basically, this plugin add a Timezone Conversion widget, which shows the date & time, from timezone and to timezone selection field on the front end of website.

Visitor can select date and time, from and to timezone from the dropdown and submission of it will show the converted date & time.

== Installation ==

Method 1:
<ol>
<li> Download the plugin tzc-timezone-conversion.zip </li>
<li> Unpack the tzc-timezone-conversion.zip file and extract the tzc-timezone-conversion folder </li>
<li> Upload the plugin folder to your /wp-content/plugins/ directory </li>
<li> Go to WordPress dashboard, click on Plugins from the menu </li>
<li> Locate the Timezone Conversion and click on Activate link to activate the plugin. </li>
</ol>

Method 2:
<ol>
<li> Download the plugin tzc-timezone-conversion.zip </li>
<li> Go to WordPress Dashboard-Plugins-Add New </li>
<li> Click on Upload Plugin link from top </li>
<li> Upload the downloaded tzc-timezone-conversion.zip file and click on Install Now </li>
<li> After installation, click on Activate Plugin link to activate the plugin. </li>
</ol>
